# 软件/App命名系统方法论

## 一、为什么之前的方法不对？

### ❌ 之前的错误方法
1. **关键词堆砌**：把"秒"、"启"、"专注"等词简单组合
2. **拟声词模仿**：盲目模仿"嘀嗒"，缺乏独特性
3. **功能直白**：过于强调功能，缺乏品牌感
4. **缺乏系统**：没有建立命名框架，随机生成

### ✅ 正确的命名思维
命名不是"想名字"，而是**建立品牌资产**。

---

## 二、命名核心法则：3法则+6公式

### 法则一：认知负荷最小化
**原则**：用户看到名字后，理解成本越低越好。

**标准**：
- 2秒内明白产品类别
- 3秒内记住品牌名称
- 5秒内产生情感共鸣

**案例**：
- ✅ 美团（美好团购）→ 2秒理解
- ✅ 滴滴（汽车鸣笛声）→ 2秒理解
- ❌ 秒启专注 → 需要解释"秒启"是什么意思

---

### 法则二：差异化记忆锚点
**原则**：在同类产品中建立独特的记忆点。

**方法**：
1. **反常识**：与行业惯例相反
2. **造新词**：创造全新的词汇
3. **借意象**：借用跨领域意象

**案例**：
- ✅ 专注森林（Forest）：用"种树"意象，而非"计时"
- ✅ 抖音：用"抖动的音乐"，而非"短视频"
- ❌ 秒启专注：与"番茄Todo"、"专注清单"同质化

---

### 法则三：情感价值预埋
**原则**：名字中预埋情感价值，让用户产生向往。

**情感类型**：
- **成就感**：Forest（种树成就感）
- **专业感**：Flow（心流状态）
- **轻松感**：Focus Dog（宠物陪伴）
- **掌控感**：滴答清单（时间管理）

**案例**：
- ✅ Forest：用户为了"种树"而专注
- ✅ Flow：用户追求"心流"状态
- ❌ 秒启专注：只有功能，没有情感

---

## 三、六大命名公式

### 公式一：组合词派（Portmanteau）
**定义**：将两个词巧妙拼接，创造新词。

**公式**：`词A + 词B = 新词`

**案例**：
- 字节跳动 = 字节（数据）+ 跳动（舞蹈）
- 美团 = 美好 + 团购
- 微信 = 微小 + 信息

**适用场景**：需要创造全新概念，建立品类认知。

---

### 公式二：意象借代派（Metaphor）
**定义**：借用跨领域意象，建立品牌联想。

**公式**：`意象 + 功能 = 品牌`

**案例**：
- Forest（专注森林）= 森林意象 + 专注功能
- 抖音 = 抖动的音乐 + 短视频
- 迅雷 = 闪电速度 + 下载

**适用场景**：需要建立强烈的品牌调性和情感连接。

---

### 公式三：拟声拟态派（Onomatopoeia）
**定义**：使用声音或状态的拟声词。

**公式**：`拟声词 + 功能/品类 = 品牌`

**案例**：
- 滴滴 = 汽车鸣笛声
- 嘀嗒 = 钟表走动声
- 哗啦啦 = 水流声

**适用场景**：需要强调速度、节奏、即时性。

**⚠️ 注意**：拟声词容易同质化，需要独特化改造。

---

### 公式四：价值主张派（Value Proposition）
**定义**：直接表达产品价值主张。

**公式**：`价值词 + 品类 = 品牌`

**案例**：
- 得到 = 获得知识
- 知乎 = 知道乎（问答）
- 懂车帝 = 懂车 + 专家

**适用场景**：需要快速建立品类认知，降低教育成本。

---

### 公式五：人格化派（Personification）
**定义**：赋予产品人格特征。

**公式**：`人格特征 + 功能 = 品牌`

**案例**：
- Focus Dog = 狗狗 + 专注陪伴
- 小冰 = 人工智能助手
- Siri = 语音助手（名字本身无意义，但人格化）

**适用场景**：需要建立情感连接，降低使用压力。

---

### 公式六：极简符号派（Minimalism）
**定义**：使用极简的符号或单字。

**公式**：`单字/符号 = 品牌`

**案例**：
- 豆瓣 = 豆（小）+ 瓣（聚集）
- 陌陌 = 陌生人社交
- Keep = 保持（运动）

**适用场景**：需要极简风格，目标用户为年轻群体。

---

## 四、专注类App命名策略

### 行业分析
当前专注类App命名存在的问题：
1. **同质化严重**：番茄、专注、清单等词过度使用
2. **功能导向**：缺乏情感价值和品牌调性
3. **缺乏想象力**：都在"时间管理"框架内思考

### 破局思路
**核心问题**：专注类App卖的不是"计时器"，而是"状态"。

**用户真正想要的**：
- 考研党："进入学习状态"
- 自由职业者："进入工作状态"
- 创作者："进入创作状态"

**命名方向**：从"工具"转向"状态"。

---

## 五、命名生成框架

### 第一步：明确核心价值
回答三个问题：
1. 用户使用后获得什么**状态**？
2. 这个状态给用户带来什么**感受**？
3. 这个感受可以用什么**意象**表达？

### 第二步：选择命名公式
根据产品定位选择：
- 要创造新概念 → 组合词派
- 要建立情感连接 → 意象借代派
- 要强调速度节奏 → 拟声拟态派
- 要降低认知成本 → 价值主张派
- 要建立陪伴感 → 人格化派
- 要极简风格 → 极简符号派

### 第三步：生成候选名单
每个公式生成5-10个候选，共30-50个。

### 第四步：筛选评估
使用评估矩阵筛选：
- 认知负荷（易理解）
- 记忆锚点（易记住）
- 情感价值（易共鸣）
- 差异化（易区分）
- 可用性（易注册）

### 第五步：测试验证
- 用户测试：目标用户是否能理解
- 搜索测试：应用商店是否可注册
- 商标测试：是否存在侵权风险

## 六、命名禁忌清单

### ❌ 绝对避免
1. **生僻字**：增加认知成本
2. **英文直译**：缺乏本土化
3. **过度抽象**：无法理解
4. **蹭热点**：缺乏长期价值
5. **过度自我**：创始人名字等

### ⚠️ 谨慎使用
1. **品类通用词**：难以建立品牌
2. **热门词汇**：容易过时
3. **地域词汇**：限制发展空间
4. **数字字母**：缺乏情感

## 七、成功案例深度解析

### 案例一：Forest（专注森林）

**命名逻辑**：
- 核心价值：帮助用户专注
- 用户状态：专注时"种树"
- 情感价值：成就感（种出森林）
- 意象借代：森林 = 专注 + 成长 + 宁静

**成功要素**：
1. ✅ 反常识：不用"计时"用"种树"
2. ✅ 情感预埋：种树成就感
3. ✅ 视觉化：森林意象易于设计
4. ✅ 国际化：Forest全球通用

---

### 案例二：抖音

**命名逻辑**：
- 核心价值：短视频娱乐
- 用户状态：抖动的音乐节奏
- 情感价值：快乐、律动
- 意象借代：抖动 = 音乐 + 视频 + 节奏

**成功要素**：
1. ✅ 差异化：不用"视频"用"抖音"
2. ✅ 情感预埋：音乐律动的快乐
3. ✅ 动作化："抖"是动词，易于传播
4. ✅ 口语化：朗朗上口

---

### 案例三：美团

**命名逻辑**：
- 核心价值：团购优惠
- 用户状态：美好生活
- 情感价值：实惠、美好
- 组合词：美好 + 团购

**成功要素**：
1. ✅ 价值主张：美好团购
2. ✅ 正向情感：美好
3. ✅ 品类暗示：团
4. ✅ 简洁易记：2字

## 八、应用到本项目

### 重新分析产品

**产品核心价值**：
- 不是"计时器"
- 不是"番茄钟"
- 是"进入专注状态的触发器"

**用户真正想要的**：
- 快速进入专注状态
- 无干扰的专注环境
- 对专注的掌控感

**关键洞察**：
用户打开App的那一刻，已经完成了"决定专注"的心理准备。
App的价值是"承接这个决定"，让用户"瞬间进入状态"。

### 命名方向

**方向一：状态触发器**
- 强调"瞬间进入状态"
- 意象：门、开关、通道、入口

**方向二：专注空间**
- 强调"无干扰环境"
- 意象：房间、岛屿、气泡、结界

**方向三：时间掌控**
- 强调"对时间的掌控"
- 意象：舵、锚、缰绳、开关

**方向四：心流状态**
- 强调"最佳体验"
- 意象：流、境、域、场

## 十一、深度推荐：入境

### 为什么"入境"是最好的选择？

**1. 认知负荷低**
- "入"：进入，动作明确
- "境"：境界、状态，含义丰富
- 2秒理解：进入专注境界

**2. 差异化强**
- 不用"专注"、"番茄"等通用词
- 不用"秒"、"启"等功能词
- 创造全新概念：专注是一种"境界"

**3. 情感价值高**
- "入境" = 进入心流状态
- 有仪式感：像进入禅定、进入心流
- 有成就感：达到一种境界

**4. 品牌延展性强**
- 产品：入境、入境Pro、入境Lite
- 功能：入境模式、深度入境、入境统计
- 社区：入境者、入境圈

**5. 国际化友好**
- 英文：Entry / Zone / Flow State
- 日文：入境（にゅうきょう）
- 韩文：입경

**6. 视觉化强**
- 门、通道、光、境界线
- 易于设计Logo和视觉系统

---

### "入境"品牌故事

**Slogan**：
- 入境，即专注
- 一秒入境，心无旁骛
- 入境，进入你的境界

**品牌调性**：
- 专业但不严肃
- 简洁但有深度
- 现代但有禅意

**用户心智**：
- 打开App = 入境
- 专注 = 入境
- 完成专注 = 出境
