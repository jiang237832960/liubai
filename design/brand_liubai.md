留白，是中国画的最高境界。

不是"没有"，而是"无限"。
于空白处，见天地，见自己。

专注，亦是如此。
不是填满每一分钟，
而是给心灵留白，
让思绪在空白处自由生长。

留白，
是专注的艺术，
是生活的智慧，
是东方美学的当代表达。

**我们相信：**
- 真正的专注，来自于心灵的留白
- 少即是多，留白创造无限可能
- 在信息过载的时代，留白是一种奢侈，更是一种必需

**我们提供：**
- 一个极简的专注空间
- 一种东方美学的专注体验
- 一次心灵的留白之旅

## 六、UI设计规范

### 1. 设计原则

#### 留白原则
- 界面留白不少于40%
- 元素间距宽松
- 内容精简，突出重点

#### 极简原则
- 每个页面不超过3个核心功能
- 去除一切装饰性元素
- 用最少的元素表达最多的信息

#### 东方原则
- 借鉴中国传统美学
- 意境深远，值得品味
- 现代演绎，不落俗套

### 2. 页面设计

#### 首页（留白页）
```
┌─────────────────────────────┐
│                             │
│                             │
│                             │
│        留 白                │
│                             │
│      [开始留白]             │
│                             │
│                             │
│                             │
├─────────────────────────────┤
│  留白  | 日志  | 设置       │
└─────────────────────────────┘
```

**设计要点：**
- 大量留白，中央放置品牌名
- 只有一个按钮：开始留白
- 底部导航极简

#### 留白中页面
```
┌─────────────────────────────┐
│                             │
│                             │
│        25:00                │
│                             │
│      留白中...              │
│                             │
│                             │
│                             │
│        [暂停]               │
└─────────────────────────────┘
```

**设计要点：**
- 全屏留白，仅显示倒计时
- 极简文字："留白中..."
- 底部一个暂停按钮

#### 留白完成页面
```
┌─────────────────────────────┐
│                             │
│                             │
│        ✓                    │
│                             │
│      留白完成               │
│      25分钟                 │
│                             │
│                             │
│    [查看日志]  [再次留白]   │
└─────────────────────────────┘
```

**设计要点：**
- 简洁的完成提示
- 显示留白时长
- 两个操作按钮

### 3. 组件设计

#### 按钮
- 圆角：4px（极简）
- 高度：48px
- 字体：16px，Medium
- 颜色：墨黑（#1A1A1A）

#### 卡片
- 圆角：8px
- 背景：白色
- 阴影：无（用留白代替）
- 边框：1px淡墨灰（#E5E5E5）

#### 输入框
- 圆角：4px
- 边框：1px淡墨灰
- 聚焦：墨黑边框
- 无阴影，无装饰
